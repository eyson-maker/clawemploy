### Highlights

- Launched the ClawEmploy Developer Agent MVP for Telegram-based task intake and execution.
- Added end-to-end agent provisioning, workspace creation, and OpenClaw registration.
- Connected agent task reporting back to the dashboard with task history and credit tracking.

### What ships in v0.1.0

- Telegram onboarding for the first Developer Agent workflow.
- Dashboard controls for hiring, waking, sleeping, and terminating agents.
- Credit-based usage controls for provisioning and completed task runs.
