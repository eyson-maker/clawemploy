### 亮点

- 发布 ClawEmploy Developer Agent MVP，支持通过 Telegram 提交任务并执行。
- 打通 Agent 创建、工作区生成与 OpenClaw 注册流程。
- 将 Agent 任务回传到仪表板，支持任务历史与 Credits 跟踪。

### v0.1.0 已上线内容

- 首个 Developer Agent 的 Telegram 接入流程。
- 仪表板中的雇用、唤醒、休眠与终止操作。
- 基于 Credits 的 Agent 创建与任务完成计费控制。
