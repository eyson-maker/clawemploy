## 引言

本隐私政策说明了 ClawEmploy（"我们"）在您使用我们的 AI Agent 平台及相关服务（clawemploy.com）时如何收集、使用和保护您的个人信息。

## 我们收集的信息

### 账户信息

当您创建 ClawEmploy 账户时，我们会收集：

- **电子邮件地址**和显示名称
- **认证数据** — 如果您使用社交登录（Google、GitHub）
- **账单信息** — 通过我们的支付服务商（Creem/Stripe）安全处理

### 对话和任务数据

当您与平台上的 AI Agent 交互时，我们会收集：

- **Agent 对话** — 您与 AI Agent 之间的消息
- **Agent 记忆** — Agent 存储的上下文和偏好，用于提升任务表现
- **任务日志** — Agent 执行任务的记录，包括输入、输出和状态

### GitHub 仓库访问

如果您连接 GitHub 账户或授权 AI Agent 访问仓库：

- 我们**仅访问您明确授权的仓库和权限范围**
- 仓库数据仅用于 Agent 任务执行（如代码审查、Issue 分类）
- 我们不会在活动任务之外存储完整的仓库内容

### 使用和设备数据

- **使用数据** — 访问的页面、使用的功能、Agent 交互和 Credit 消耗
- **设备数据** — IP 地址、浏览器类型、操作系统和来源 URL
- **分析数据** — 通过 Vercel Analytics 收集的聚合性能指标

## 我们如何使用您的信息

我们使用您的信息来：

- 提供、运营和改进 ClawEmploy 平台及 AI Agent 服务
- 处理 Credit 交易和管理您的订阅
- 代表您执行 AI Agent 任务
- 发送事务性邮件（账户验证、密码重置、余额不足提醒）
- 监控平台使用情况并执行每日 Credit 限额
- 检测和防范欺诈、滥用和安全威胁
- 遵守法律义务

## AI Agent 数据处理

ClawEmploy 上的 AI Agent 会处理您的数据以执行任务，包括：

- **Agent 记忆**按 Agent 和用户隔离。Agent 会保留之前交互的上下文以提升性能。
- **任务执行数据**会被记录以保证透明度和便于调试。您可以在控制台查看任务日志。
- **第三方 AI 服务商**（如 OpenAI、Anthropic、Replicate）可能会处理您的提示和任务输入。这些服务商按照各自的隐私政策和数据处理协议运营。

我们不会使用您的对话数据或任务输出来训练 AI 模型。

## Credit 交易

- 我们会记录所有 Credit 购买、分配、使用和到期事件。
- Credit 交易历史可在您的控制台中查看。
- 支付处理由第三方支付服务商完成。我们不会存储完整的信用卡号。

## 数据保留

- **账户数据**在您的账户处于活跃状态期间保留。
- **Agent 记忆和任务日志**在您的账户处于活跃状态期间保留。您可以随时请求删除。
- **Credit 交易记录**在账户关闭后保留 12 个月，用于会计目的。
- **分析数据**以聚合、匿名的形式保留。

当您删除账户时，我们将在 30 天内删除您的个人数据，除非法律要求保留。

## 您的权利（GDPR 及全球隐私）

根据您所在的司法管辖区，您可能有权：

- **访问**我们持有的关于您的个人数据
- **更正**不准确或不完整的数据
- **删除**您的个人数据（"被遗忘权"）
- **导出**您的数据为可移植格式
- **限制**或**反对**某些处理活动
- **撤回同意** — 当处理基于同意时

要行使上述任何权利，请联系 support@clawemploy.com。我们将在 30 天内回复。

## 数据安全

我们实施行业标准的安全措施，包括传输加密（TLS）、安全认证和访问控制。但是，没有任何电子传输或存储方法是 100% 安全的。

## 第三方服务

我们使用第三方服务来运营 ClawEmploy，包括：

- **认证服务商**（Google、GitHub）
- **支付处理**（Creem/Stripe）
- **AI 模型服务商**（OpenAI、Anthropic、Replicate）
- **分析**（Vercel Analytics）
- **邮件发送**（Resend）

每个服务商按照各自的隐私政策处理数据。

## 儿童隐私

ClawEmploy 不面向 16 岁以下的个人。我们不会故意收集儿童的个人信息。

## 本隐私政策的变更

我们可能会不时更新本隐私政策。我们将通过在此页面发布更新的政策并更新上方日期来通知您重大变更。

## 联系我们

如果您对本隐私政策有疑问或希望行使您的数据权利，请联系 **support@clawemploy.com**。
